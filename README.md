# Code-Quiz

A sample timed quiz based mainly on Javascript and Bootstrap.

## How it works
- Start Quiz
- Timer set to 75
- Each wrong answer [ -10seconds ]
- Scorepage that shows & saves your results
- Clear-able Scoreboard page
